Description

This module allows users to create promotion rules & items first. They are then used in the promotion programs so that they can be reusable. Several promotions rule types and several promotion item types such as Fixed Discount, Percentage Discount, Single Free Product, Multiple Free Products with specific quantities can be configured. Promotion programs to be applied in an order can be selectable.

Key Features

Promotion rules & items can be created separately.
Promotion rules & items that are reusable in multiple promotion programs.
Product wise/Category wise promotion rules & items.
Creating programs that give promotion items when customers buy combo products.
Giving multiple set of free products.
Defining maximum discount amount & merging all discounts in one single product.
